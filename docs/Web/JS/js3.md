## 格式化字符串

+   字符串中填入数值：

```js
let name = 'yxc', age = 18;
let s = `My name is ${name}, I'm ${age} years old.`;
```

+   定义多行字符串：

```js
let s = 
`<div>
    <h2>标题</h2>
    <p>段落</p>
/div>`
```

+   保留两位小数如何输出

```js
let x = 1.234567;
let s = `${x.toFixed(2)}`;
```
