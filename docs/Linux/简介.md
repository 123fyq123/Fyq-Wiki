## 概述

**Linux 是一种自由和开放源码的类 UNIX 操作系统。**

Linux 是在 1991 由林纳斯·托瓦兹在赫尔辛基大学上学时创立的，主要受到 Minix 和 Unix 思想的启发。

这章，将为大家介绍如何使用 Linux。

目前市面上较知名的Linux发行版有：Ubuntu、RedHat、CentOS、Debian、Fedora、SuSE、OpenSUSE、Arch Linux、SolusOS 等。
